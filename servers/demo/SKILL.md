# Demo Skill

This is a sample skill created to demonstrate how to add new skills to the agent system.

## Tools

### greet
Returns a personalized greeting.
- `name`: The name of the person to greet.

### add_numbers
Adds two numbers and returns the sum.
- `a`: First number.
- `b`: Second number.
